Wine Runner Suite

Effortlessly run Windows applications on your Debian-based Linux system!

Quick Install / Update

Run the following command in your terminal. It will download and install the latest stable version of Wine Runner Suite:

curl -sSL https://tapelu-io.github.io/install.sh | sudo bash

Note: This command requires sudo privileges to install the package and its dependencies system-wide. You will be prompted for your password if necessary. The script will attempt to install dependencies like Wine (from WineHQ if possible), Winetricks, etc.

To install a specific version (e.g., v1.1.3, check available tags on the releases page):

curl -sSL https://tapelu-io.github.io/install.sh | sudo bash -s -- --version v1.1.3

What is Wine Runner Suite?

Wine Runner Suite simplifies running Windows applications and games on Debian, Ubuntu, and their derivatives. It provides:

  • A unified wine-runner command-line tool.
  • Automatic Wine prefix initialization with essential components like corefonts, vcrun2022, and an attempt to install Microsoft Edge WebView2.
  • Easy installation of .exe and .msi files with an option to auto-run the application post-install.
  • An offer to create Linux desktop shortcuts for installed Windows applications.
  • Configuration via /etc/wine-runner/wine-runner.conf for CPU affinity and extra Winetricks verbs.
  • Seamless double-click integration for .exe and .msi files.

Basic Usage

Once installed, you can use it from the command line:

Run an existing application:

wine-runner /path/to/your/application.exe

Install an application (and attempt to run it + create shortcut):

wine-runner --install /path/to/your/setup.exe

wine-runner --install /path/to/your/installer.msi

Alternatively, just double-click .exe or .msi files in your file manager!

For more options, run:

wine-runner --help
